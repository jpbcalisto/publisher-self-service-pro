📋 COMO TESTAR A APLICAÇÃO

🖱️  DUPLO-CLIQUE no ficheiro "EXECUTAR.command"
    → O browser abre automaticamente
    → A aplicação está pronta para testar

⚠️  REQUISITOS:
    → macOS com Python 3 instalado
    → Browser moderno (Chrome, Safari, Firefox)

🔧 FUNCIONALIDADES PARA TESTAR:
    ✅ Email: deve ter @ e . (ex: teste@site.com)
    ✅ Telefone: só números e + opcional (ex: +351912345678)
    ✅ URL: formato domain.extension (ex: exemplo.com)
    ✅ GEO: "Other" completa automaticamente para 100%
    ✅ Traffic: Desktop/Mobile completam para 100%
    ✅ Formatos aparecem com nomes legíveis no final

❌ PARA PARAR: Feche a janela do terminal
